Everything you need to analyse any stock

Four engines. One command. Instant results.

P

Piotroski F-Score

9-signal financial health check. Profitability, leverage, liquidity, efficiency — all scored in one number out of 9.

Z

Altman Z-Score

Bankruptcy risk prediction using Altman's proven formula. Know if a company is in the safe zone before you invest.

M

Beneish M-Score

8-variable earnings manipulation detector. Catch accounting red flags before they become front page news.

XL

Excel Reports

Every scan generates a formatted multi-sheet Excel workbook with charts, color coding, and a Dashboard sheet.

UI

HTML Dashboard

Interactive browser dashboard with gauges, radar charts, valuation benchmarks. Print to A4 in one click.

MCP

MCP Server

Connect telmus to Claude, Cursor, or Windsurf as an MCP tool. Ask your AI IDE to analyse any stock in plain English.

From command to dashboard in seconds

1

Install

One command. Python 3.9+ required. No API key needed.

pip install telmus
2

Scan any stock

Works with 50,000+ global tickers. Add .NS for Indian NSE stocks.

telmus scan AAPL telmus scan INFY.NS telmus scan RELIANCE.NS
3

Get instant analysis

Piotroski F, Altman Z, Beneish M, valuation ratios, growth metrics, analyst brief — all in seconds.

# Terminal shows full analysis table
4

Open your dashboard

TICKER_analysis.xlsx and TICKER_dashboard.html open automatically. Print the dashboard as a one-page A4 report.

# Excel + HTML auto-generated

Six powerful commands

Everything you need for fundamental analysis

telmus scan INFY.NS telmus scan AAPL telmus scan RELIANCE.NS --json

Runs full analysis: valuation, health, growth, and flags engines. Generates analysis workbook and interactive dashboard instantly.

INFY_analysis.xlsx INFY_dashboard.html
telmus compare INFY.NS TCS.NS telmus compare AAPL MSFT

Side by side comparison of two stocks across all metrics. See which company wins on each metric with color coding.

INFYvsTCS_comparison.xlsx INFYvsTCS_dashboard.html
telmus screen --sector IT telmus screen --sector Banking --min-piotroski 6 telmus screen --sector IT --max-de 1.5

Scan an entire sector at once with custom filters. Find the strongest stocks in any sector instantly.

screen_results.xlsx screen_dashboard.html
telmus check INFY.NS telmus check AAPL

Quick health check only. Just Piotroski F and Altman Z. Fastest way to check if a stock is worth deeper analysis.

telmus powerbi --tickers INFY.NS TCS.NS WIPRO.NS --export portfolio.csv

Generates Power BI ready CSV for portfolio analysis. Also creates portfolio_report.html with pre-built charts.

portfolio.csv portfolio_report.html
telmus serve

Starts MCP server for Claude Desktop, Cursor, or Windsurf. Add to your AI IDE config and ask it to analyse stocks in plain English.

{ "mcpServers": { "telmus": { "command": "telmus", "args": ["serve"] } } }
