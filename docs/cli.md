# CLI Reference

Commands:

- `valkit scan TICKER`: Run a full scan.
- `valkit scan TICKER --json`: Output raw JSON.
- `valkit scan TICKER --export FILE.json`: Save scan output.
- `valkit compare TICKER_A TICKER_B`: Compare two tickers.
- `valkit screen`: Run a simple screener.
- `valkit serve`: Start the MCP server.
- `valkit info`: Show version and data source.
- `valkit check TICKER`: Quick health check.
